ESP32 BLE Gaming Controller

A custom Bluetooth Low Energy (BLE) gaming controller developed using the ESP32 platform. The system integrates dual analog joysticks, D-Pad controls, trigger buttons, motion sensing using MPU6050, RGB LED feedback, battery monitoring and wireless BLE HID connectivity.

View GitHub Repository

Project Overview

This project was developed as part of the Electronic System Design Project Lab (Spring Semester 2026) at IIT Hyderabad. The objective was to design a fully functional wireless gaming controller from the ground up, including schematic design, PCB layout, embedded firmware, BLE communication, motion sensing integration and battery management. The final system functions as a Bluetooth HID gamepad compatible with Windows, Linux and Android devices.
